# Smart Task Manager

## Project Overview
The Smart Task Manager is designed to streamline task management for users, enhancements being on  intuitive design features for productivity and task completion.

## Features
- **Task Creation**: Users can easily create and organize tasks with deadlines and priorities.
- **Task Scheduling**: The system allows for scheduling of tasks based on user preferences.
- **Notifications**: Users receive reminders for upcoming tasks and deadlines.
